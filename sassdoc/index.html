    General

    mixins

    flex--column

    @mixin flex--column() { ... }

    Description

    Flex items laying out in vertical columns

    Parameters

    None.

    Example

    Using

    .foo {
  @include flex--column;
  justify-content: flex-end;
}

    Output CSS

    .foo {
  display: flex;
  flex-direction: column;
  justify-content: flex-end;
}

    Author

    • Radmir Nabiullin

    placeholders

    list

    %list { ... }

    Description

    List without padding and symbols before list items

    Example

    Using

    .foo {
  @extend %list;
}

    Output CSS

    .foo {
  padding: 0;
  list-style: none;
}
